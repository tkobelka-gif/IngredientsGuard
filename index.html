⚠️ Cannot connect to API — hosting required

You're opening this file directly from storage (file:// origin). iOS browsers block all outbound API calls from local files as a security policy — the Anthropic API cannot be reached this way, regardless of which browser you use.

Fix: host the file for free on GitHub Pages (5 min setup)

  1. Create a free account at github.com
  2. Click + → New repository. Name it anything, set to Public, click Create.
  3. Click Add file → Upload files, upload this ingredient-guard-v4.html file.
  4. Rename it to index.html (click the file → pencil icon → change name at top).
  5. Go to Settings → Pages, set Source to main branch / root, click Save.
  6. GitHub gives you a URL like https://yourname.github.io/reponame — open that on your iPhone. Done! ✓
Alternative: If you have a Mac on the same Wi-Fi, run python3 -m http.server 8080 in the folder containing this file, then open http://YOUR-MAC-IP:8080 in Edge on your iPhone.
